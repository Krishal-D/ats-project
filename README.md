# ATS Project (Applicant Tracking System)

This repository contains a full-stack Applicant Tracking System (ATS) scaffold.

Tech stack (simplified):

- Frontend: React.js (Create React App) + CSS
- Backend: Node.js + Express
- Database: PostgreSQL

This scaffold contains starter files and instructions to get you started quickly.

Quick start (local):

1. Clone this repo (or initialize locally)
2. cd backend && npm install
3. cd frontend && npm install
4. Start backend and frontend according to README files

See the `backend/README.md` and `frontend/README.md` for more details.
